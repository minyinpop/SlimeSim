## 🔀 分支簡介 🔀
  * master 分支裡面為 `遊戲執行檔`，可供直接試玩
   
  * unity 分支裡面為 `Unity 的資產包`，歡迎取用 :D
    * .unitypackage 為 `Unity 編輯器的資產包`，直接把它拉進 Assets 裡面即可
    * Scene 資料夾是 `遊戲場景`，一樣拉進 Assets 裡面即可
    
## 💡 遊玩操作 💡
* **A or D / 左右移動**
* **W or Space / 跳躍**
* **E / 互動**
* **C / 合成介面**
* **F / 手電筒**
